# 📋 Task Manager Android

Application Android permettant de gérer ses tâches quotidiennes.

## 🎯 Objectif

Aider les utilisateurs à organiser leurs activités et améliorer leur productivité.

## ✨ Fonctionnalités

- Ajouter une tâche
- Modifier une tâche
- Supprimer une tâche
- Marquer comme terminée
- Liste des tâches
- Historique

## 🛠️ Technologies

- Kotlin
- Android Studio
- Room Database
- MVVM

## 📚 Compétences démontrées

- CRUD Android
- Base de données locale
- Architecture MVVM
- Gestion d'état

## 📱 Écrans

- Splash Screen
- Liste des tâches
